## 🚀 Como Publicar no GitHub

### Passo 1: Criar o repositório
1. Acesse [github.com](https://github.com)
2. Clique em **New repository**
3. Nomeie como `jardim-digital` (ou outro nome)
4. Deixe como **Public**
5. Clique em **Create repository**

### Passo 2: Enviar os arquivos
No terminal, dentro da pasta do projeto:

```bash
git init
git add .
git commit -m &quot;Primeiro commit - Jardim Digital&quot;
git branch -M main
git remote add origin https://github.com/SEU_USUARIO/jardim-digital.git
git push -u origin main
```

### Passo 3: Habilitar GitHub Pages
1. Vá em **Settings** do repositório
2. Role até **GitHub Pages**
3. Em **Source**, selecione **main** e **/(root)**
4. Clique **Save**
5. Aguarde ~1 minuto e seu site estará no ar!

O link será algo como: `https://SEU_USUARIO.github.io/jardim-digital`

## ✨ O que o site tem

- **Tela inicial** com mensaje no estilo dela (affs)
- **11 rosas** representando os 11 meses juntos
- **Animações suaves** - flores aparecem aos poucos com fade-in
- **Flores flutuam** levemente esperando clique
- **Ao clicar** - pétalas se abrem e revelam uma mensagem
- **Céu gradiente** estilo pôr do sol com estrelas
- **Música** toca em background (low volume)
- **Mensagem final** aparece quando todas são clicadas
- **Responsivo** - funciona bem no celular

- Gradiente de azul para rosa (céu ao entardecer)
